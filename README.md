# 🏃 Move Now

App móvil diseñada para combatir el sedentarismo en Bogotá y el mundo.
Ayuda a las personas a mantenerse activas con rutinas diarias, recordatorios
inteligentes y un sistema de rachas motivacional.

## ✨ Características

- 🔔 Recordatorios diarios de movimiento
- 🏋️ Rondas de ejercicio con niveles de esfuerzo personalizables
- 🔥 Racha diaria para mantener la motivación
- 📊 Ponderado de niveles adaptable al usuario
- ✅ Meta diaria con seguimiento de cumplimiento

## 🎯 Objetivo

Lograr que personas con estilos de vida sedentarios puedan acceder
fácilmente a rutinas de ejercicio saludables y sostenibles en el tiempo.

## 🚀 Tecnologías

- Java (Android)
- Android Studio
- Gradle
- Firebase Firestore (base de datos)
- Firebase Cloud Messaging (notificaciones)
- Navigation Component
- Material Design 3
